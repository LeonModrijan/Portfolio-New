# Leon Modrijan | Portfolio

Modern, glassmorphic portfolio focused on clean UI and smooth user experience.

[View Live Demo](https://leonmodrijan.online)

---

### 🚀 Key Features
* **Glassmorphism UI** – Modern semi-transparent backgrounds.
* **Responsive** – Optimized for all devices (Web & Mobile).
* **Performance** – High-speed loading using `.webp` assets.
* **Custom UX** – Custom cursor and smooth scroll navigation.
